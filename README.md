# app-rastreio

**Aplicativo Desktop para rastreamento de objetos dos correios.**

```bash
# Clone this repository
git clone https://github.com/lazaroness/app-rastreio.git
# Go into the repository
cd app-rastreio
# Install dependencies
npm install
# Run the app
npm start
```
